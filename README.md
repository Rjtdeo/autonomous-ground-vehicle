# Multi-Sensor Autonomous Ground Vehicle
### ROS2 | ESP32 | Arduino | Raspberry Pi | MQTT | 24GHz Radar | GPS | IMU

A fully autonomous campus navigation rover built as an M.S. AI thesis project at Florida Atlantic University. Mirrors real-world ADAS architecture — distributed sensing, publish-subscribe communication, and autonomous decision-making.

## Hardware
- Raspberry Pi 3B+ (ROS2 brain)
- ESP32 (sensor hub + WiFi)
- Arduino UNO R4 (PWM motor control)
- 24GHz mmWave Radar (HLK-LD2450)
- 3x HC-SR04 Ultrasonic sensors
- MPU6050 IMU, NEO-6M GPS

## Stack
ROS2 Humble | MQTT | Python | Arduino C | Ubuntu 22.04
